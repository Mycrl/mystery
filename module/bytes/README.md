A utility library for working with bytes, bring heap allocation to node.js.


## Usage

to use `bytes`, first add this to your `package.json`:
```
"dependencies": {
    "bytes": "file:./bytes"
}
```

next, add this to your project:
```
import { Bytes, BytesMut } from 'bytes'
```
